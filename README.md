## Selected Projects

### AirPipe - QR Code File Transfer
*Transfer files between devices by scanning a QR code in the terminal.*
*   **Technologies:** Go, WebSocket, NaCl, Docker
*   E2E encrypted relay (zero-knowledge, only sees ciphertext). Multi-file support with auto-zip, curl/wget-compatible short URLs.
*   Open source, 50 GitHub stars, featured on r/selfhosted.
*   [Github](https://github.com/Sanyam-G/Airpipe)

### Drift - Live Multiplayer Cursors
*Visitors on my website see each other's cursors moving in real time.*
*   **Technologies:** Go, WebSocket, JavaScript, Docker
*   GeoIP country detection, flag emoji cursors, session recording, ghost replay of past visitors.
*   1,600+ LinkedIn impressions on launch.
*   [Live](https://cursors.sanyamgarg.com) | [Github](https://github.com/Sanyam-G/Drift)

*   **Technologies:** Python, FastAPI, Docker, SQLite, Leaflet.js
*   Parses syslog, geolocates attacker IPs, serves live data via REST API. Processes 25,000+ attacks daily.
*   [Live](https://sentinel.sanyamgarg.com) | [Github](https://github.com/Sanyam-G/Sentinel)
